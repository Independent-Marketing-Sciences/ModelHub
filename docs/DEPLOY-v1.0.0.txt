  DEPLOY v1.0.0 TO GITHUB RELEASES - QUICK REFERENCE
═══════════════════════════════════════════════════════════════════

Repository: https://github.com/Independent-Marketing-Sciences/ModelHub

───────────────────────────────────────────────────────────────────
STEP 1: Get GitHub Token
───────────────────────────────────────────────────────────────────

1. Go to: https://github.com/settings/tokens

2. Click: "Generate new token (classic)"

3. Set:
   Name: ModelHub Release Publisher
   Expiration: No expiration
   Scopes: ☑ repo (Full control)

4. Click "Generate token"

5. COPY THE TOKEN (looks like: ghp_xxxxx...)


───────────────────────────────────────────────────────────────────
STEP 2: Set Token in Terminal
───────────────────────────────────────────────────────────────────

Windows CMD:
  set GH_TOKEN=ghp_your_token_here

Windows PowerShell:
  $env:GH_TOKEN="ghp_your_token_here"

Verify:
  echo %GH_TOKEN%


───────────────────────────────────────────────────────────────────
STEP 3: Build and Publish
───────────────────────────────────────────────────────────────────

Run this command:

  npm run electron:build:win -- --publish always

Wait: 10-15 minutes ☕

Success when you see:
  ✓ building target=nsis
  ✓ publishing file=Modelling Mate Setup 1.0.0.exe
  ✓ published


───────────────────────────────────────────────────────────────────
STEP 4: Verify Release
───────────────────────────────────────────────────────────────────

Check: https://github.com/Independent-Marketing-Sciences/ModelHub/releases

Should see:
  ✓ Release v1.0.0
  ✓ Modelling Mate Setup 1.0.0.exe
  ✓ latest.yml


───────────────────────────────────────────────────────────────────
STEP 5: Share with Users
───────────────────────────────────────────────────────────────────

Send users this link:
  https://github.com/Independent-Marketing-Sciences/ModelHub/releases/latest

They download and install the .exe file.

AUTO-UPDATES WORK AUTOMATICALLY! 🎉


───────────────────────────────────────────────────────────────────
FUTURE UPDATES (v1.0.1, v1.1.0, etc.)
───────────────────────────────────────────────────────────────────

1. Edit package.json version number

2. Update CHANGELOG.md

3. Run: npm run electron:build:win -- --publish always

4. Users get automatic update notification!


───────────────────────────────────────────────────────────────────
TROUBLESHOOTING
───────────────────────────────────────────────────────────────────

Error: "GitHub token is not set"
  → set GH_TOKEN=your_token

Build fails:
  → npm install
  → Try again

Token issues:
  → Regenerate token with 'repo' scope
  → Set token again
